1. Introdução ao HTML

1.1 Conceito

HTML é a abreviatura de Hipertext Markup Language. Trata-se de uma linguagem de marcação de hipertexto que é interpretada pelos navegadores. Normalmente melhor interpretada pelo Chrome , Firefox, Safari etc, exceto Internet Explorer.

A exibição do conteúdo no navegador é realizada por meio de tags, que são elementos que delimitam os componentes de uma página Web.
